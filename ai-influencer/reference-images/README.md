## Rules

1. **Never delete images.** Even bad generations — they help compare drift.
2. **Always upload `mira-face-only.png` as character reference** before every generation.
3. **If Mira's face starts drifting** across generations, regenerate from `mira-canonical.png` and re-crop `mira-face-only.png`.
4. **Back these up.** If you lose these files, you lose Mira. Drop a copy into Google Drive or similar.
